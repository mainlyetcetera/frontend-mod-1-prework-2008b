logical operators
&& and ||
operators that work based on boolean logic

```javascript
'foo' && 'bar'; // 'bar'
'bar' && 'foo'; // 'foo'
'foo' && ''; // '';
'' && 'foo'; // '';

'foo' || 'bar'; // 'foo'
'bar' || 'foo'; // 'bar'
'foo' || ''; // 'foo'
'' || 'foo'; // 'foo'
```
